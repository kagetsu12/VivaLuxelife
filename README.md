# VivaLuxelife - A Timeless Lifestyle Blog

A beautiful, vintage-inspired lifestyle blog website featuring curated content across six categories: Fashion & Accessories, Health & Beauty, Home & Garden, Travel & Accommodation, Finance & Insurance, and Food & Beverage.

## Features

- **Vintage-Inspired Design**: Elegant, retro aesthetic with paper textures and classic typography
- **Six Content Categories**: Comprehensive lifestyle coverage
- **Article Management**: 5 detailed blog articles with images
- **Product Recommendations**: Curated product suggestions for each article
- **Search Functionality**: Search articles by title or content
- **Category Filtering**: Filter articles by category
- **Pagination**: Easy navigation through articles
- **Responsive Design**: Works on desktop, tablet, and mobile devices

## How to Use

1. **Open the Website**: Simply open `index.html` in your web browser
2. **Navigate**: Use the top navigation menu to browse categories, read articles, or visit About/Contact pages
3. **Search**: Use the search box to find articles by keywords
4. **Filter**: Click on category links to filter articles
5. **Read Articles**: Click on any article card to read the full article with product recommendations

## Design Features

- **Color Palette**: Warm browns, creams, and deep blues
- **Typography**: Playfair Display (headings), Dancing Script (navigation), Crimson Text (body)
- **Visual Effects**: Paper textures, subtle shadows, smooth transitions
- **Images**: High-quality Unsplash images with vintage-inspired filters

## Browser Compatibility

This website works best in modern browsers:
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Notes

- This is a static website - no server required
- All images are loaded from Unsplash CDN
- JavaScript is used for search, filtering, and pagination functionality
- The contact form provides client-side validation (for a production site, you would need a backend)

## Customization

To customize the website:
- Edit `css/style.css` for styling changes
- Modify `js/main.js` to add more articles or change functionality
- Update HTML files to change content or structure
